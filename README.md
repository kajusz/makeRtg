# makeRtg

Script for making rating tags from a png

Usage: ./makeRtg.py <pathToPngFile>
   or just drag and drop the png file onto this script

Depends: Python3, OpenDCP, ffmpeg/libav, PIL/Pillow (Optional)

You need:
* fonts:
  * Rockingham Bold
  * Rockingham Regular
* templates:
  * bbfc_scope.psd
  * bbfc_flat.psd
